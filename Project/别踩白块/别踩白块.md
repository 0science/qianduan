# 开发准备
## 开发背景

《别踩白块儿》最初是由Umoni Studio制作的一款休闲益智游戏。该游戏于2014年4月16日上线App Store。游戏上线后，在全球40多个国家和地区的免费游戏榜中登顶榜首，上线3个月累计下载量达1亿，由此可见该游戏的受欢迎程度。本章将通过JavaScript脚本语言来模拟这个小游戏。
